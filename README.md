# Upward Art / 向上管理的艺术

> 代码本身不重要，变成 PPT 才重要。

程序员向上管理话术库——收集、整理各种把技术问题翻译成领导听得懂（甚至觉得高大上）的语言的案例。

灵感来源于 Bilibili UP 主「老头们的快乐生活」(seko 阿色) 制作的一系列讽刺短剧。每集都在讲同一个故事：**程序员遇到的技术问题、职场困境或管理荒诞，经过包装翻译，最终变成领导口中的体系、战略和哲学**。

## 核心结构

```
技术真相 → 包装话术 → 领导认知 → PPT 呈现
```
