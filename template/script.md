整理脚本和原文时，数字写法遵循以下规则（可不与 ASR 原文完全一致，以可读性优先）：

1. 技术参数统一用缩写：4K、800MB、1.8TB、99.7%、91.2%
2. 大数用"阿拉伯数字 + 万/亿"：86万条、1.8TB，不写"八十六万条""一点八T"
3. 百分比用阿拉伯数字 + %：99.7%，不写"百分之九十九点七"
4. 对话部分可保留少量口语化中文数字（如"两千来个""五千年"），但技术参数仍用阿拉伯数字
