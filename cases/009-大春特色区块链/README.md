# 大春特色区块链

> 王处看到隔壁老赵讲区块链得到大领导竖大拇指，要求两周内造一套"比老赵好十倍"的区块链。旭阳把操作日志贴到审批界面下方，加了锁头图标和 MD5 哈希，王处现场领悟后发明了"有领导的区块链""低碳区块链""大春特色区块链"三个概念，20 页 PPT 预定，老赵 18 页的舞台直接被碾压。

## 技术真相

旭阳在审批流程页面每条记录下方加了一排灰色小字，按时间倒序显示操作日志，每行前面加了个 CSS 锁头图标。"链式结构"是把上一条记录的 ID 拼上时间戳跑个 MD5，存在下一条记录的一个字段里。本质就是操作日志加了个好看的界面，用 MD5 做了个伪链式校验。372 笔审批记录全在上面，但没有任何真正的分布式节点、共识机制或加密保障。

## 领导话术

- 你叫它日志，它就是日志；你叫它可信溯源链，它就是可信溯源链
- 我们的区块链有中心、有领导、有组织，像一条铁链；外面的去中心化像一盘散沙
- 传统区块链烧一个三线城市的电费，我们一个数据库搞定，零碳、绿色、低碳区块链
- 外面的区块链出了事儿找谁？找矿工？在冰岛？在蒙古？我们这个有中心，中心就是我，王大春出了事儿我负责
- 审批流程不可篡改，这是铁律
- 大春特色区块链——外国人的区块链去中心化像一盘散沙，我们的区块链有中心、有领导、有组织，像一条铁链
- 区块链的本质是什么？就四个字：有迹可循
- 老赵讲概念，我讲落地；老赵画饼，我端菜
