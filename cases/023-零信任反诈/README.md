# 零信任反诈

> 浩宇把 mock 数据"我是你爹"跟着代码推上生产，王处亲自测报名缴费时收款方显示这五个字，截图兴师问罪。旭阳五分钟内急中生智，把 mock 数据包装成"反钓鱼动态暗号"——服务器实时下发、每小时轮换、骗子仿不了。王处当场实测整点自动换号后信服，把这套"零信任反诈动态暗号体系"写进年度反诈工作亮点，规划从支付扩展到登录、短信、客服全渠道，方案名"天王盖地虎"。

## 技术真相

浩宇连调支付回调接口时，甲方还没给接口，他用 mock 数据顶着，被王处需求气得肝痛，在 mock 的收款方名称里填了"我是你爹"泄愤。本来想本地看一眼就删，核代码时没看这块，直接上线了。支付成功页收款方显示"我是你爹"。王处自己拿手机测报名缴费时发现了，截图后怒气冲冲来兴师问罪。

旭阳五分钟内的应对：①让九如紧急配一套暗号词库和每小时轮换定时任务，把"我是你爹"留在词库里；②向王处解释这是"反钓鱼动态暗号"——服务器实时下发、每小时轮换，骗子仿不出因为连不上服务器；③让王处盯着时钟整点验证自动轮换，三人不碰键盘；④对"我是你爹"这个词，承认词库审核有漏洞，但翻转为"是您亲手测出来帮我们做了最真实的线上验收"。

最终方案：①暗号扩展到登录、短信、客服、APP 开屏全渠道；②王处定名"天王盖地虎：零信任动态暗号反诈体系"，旭阳配副标题"宝塔镇河妖"。
