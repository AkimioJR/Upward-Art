# 难用是门槛

> 五年前用 3GB 搭的查询后台，只能用 IE6、手输八位数日期、查一次转十几秒，上线第三天就上了等保，五年没人敢动它。邻省第五批考察组要来，王处禁止改一个字："落灰才显得它有年头。"系统难用不是缺点是门槛，锦旗 11 面是招牌，七级台阶上每一级都有人靠它领功——谁戳穿谁就是跟所有人为敌。牌子不能重构，你一重构牌子就掉了。

## 技术真相

五年前用 3GB 搭的一个查询后台（"一展通"），只能用 IE6，日期要手输八位数，查一次转十几秒。上线第三天就上了等保，从此五年没人敢碰它。下面的办事员查一条记录要点九次、填三个八位数日期，错一个就从头来，宁可打电话问原单位也没人用。系统 173 条投诉，没人用但天天开着——"它跑就行"。

邻省第五批考察组要来，加媒体做上线五周年专题。浩宇想趁机优化，被王处制止："系统一个字儿别动，万一改出个 bug，后天当场崩了，这五年全完了。落灰才显得它有年头，经得起时间考验。"

王处的部署：①专题片五分钟，航拍+纪录片配乐+数字弹出+浑厚旁白；②考察组不许上手操作，只看锦旗听讲解；③旭阳提出挂留言墙，让来访单位签名，后来者看到"这么多人都来过"自动产生信任——"口碑势能差"；④数据往大了凑，反正没人核。

## 领导话术

- 系统好不好用不重要，能跑就行。牛必须吹出去
- 难用是门槛。你把它做得跟微信一样简单，领导一看"谁都能做嘛"，不值钱了。正因为它难，领导才觉得专业
- 你把它做简单了，你就把自己做没了
- 这不是利益共同体，这叫荣誉共同体。大家一起把这块牌子扛起来，谁也别拆谁的台
- 一个东西只要有足够多的人需要它是真的，它本身是什么早就不重要了
- 这个故事从我嘴里出去，爬了七级台阶，每一级都有人靠它领了一次功。谁敢戳穿，谁就是跟这七级台阶上所有人为敌
- 落灰才显得它有年头，经得起时间考验
- 牌子不能重构，你一重构牌子就掉了
- 片子是我自己讲，这墙是让别人替我讲，别人讲比我自己讲可信一百倍
- 口碑势能差——墙越高，落差越大，信任从高处往低处流，到最后不用开口
