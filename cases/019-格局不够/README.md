# 格局不够

> 小孙想跟张局实话实说"项目完不成"，被王处叫停，亲授向上管理四招：报喜不报忧（造喜）、选择题不要问答题（把话筒递给领导、把雨衣留给自己）、揣摩没说的部分、领导永远是对的。小孙现场实战汇报，张局满意点头。

## 技术真相

项目百分百完不成——人手不够、预算不够、时间不够。小孙打算跟张局实话实说，让领导"早点有心理准备"。王处认为这是"格局不够"，领导要的不是问题，是答案。于是传授四招向上管理心法：

1. **报喜不报忧，没有喜就造喜**：人不够 → "精简人力配置攻坚克难"；钱不够 → "有限资源投入产出比最大化"
2. **永远给选择题，不给问答题**：不问"项目咋办"，而是"方案 A 稳妥、方案 B 高效，您倾向哪个？"——干好了是领导决策，砸了是执行偏差
3. **揣摩没说的部分**：领导说"随便吃点"，意思是你来安排但不能真随便
4. **领导永远是对的**：领导说天晴外面下雨，你说"您看得远，雨一会儿就停"

实战演练后，小孙用这套话术向张局汇报，获得认可。

（视频中间穿插蓝盒子床垫广告，王处借"做事儿的格局"之名植入品牌信息。）

## 领导话术

- 不是人不够，是精简配置攻坚克难，展现高度的责任担当
- 不是钱不够，是有限资源约束下实现投入产出比最大化
- 不是完不成，是过程中遇到需要更高层面统筹的问题
- 领导要的不是问题，是答案；不是问答题，是选择题
- 干好了是领导英明决策，砸了是执行层面出了偏差
- 把功劳的话筒递给领导，把责任的雨衣留给自己
- 忠言得裹上糖衣，一个掉脑袋一个升官，差在糖衣
- 领导永远是对的，是你理解的角度还不够全面
