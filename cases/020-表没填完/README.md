# 表没填完

> 事业编报名系统：38 个必填项、40 分钟的表、15 分钟 session 超时、没有保存键，173 条投诉全是骂的。王处把 bug 包装成"多维度人才素质评估矩阵"——填五遍的是将才，投诉是筛选结果不是 bug，昨晚就把报告题目写好揣内兜了。173 条投诉不是危机，是弹药。

## 技术真相

事业编报名系统有 38 个必填项加上传附件，正常人填完需要 40 分钟。但 session 过期时间是 15 分钟（王处当初定的，理由是"安全第一"）。保存草稿功能被王处砍了（"一个表填完提交就行了，搞那么复杂干什么"）。结果：理论上没有任何正常速度的人能在超时之前填完。用户填到第 37 项时登录过期，全部白填。

收到 173 条投诉，全是骂的。浩宇建议加保存键、延长超时时间。王处拒绝，反而把 bug 包装成人才筛选机制：超时 = 数字时代的科举关门，没有保存键 = 不给你马骑你走着来。填三遍成功 = 有韧性有策略，填五遍成功 = 千军万马中杀出来的将才。173 条投诉者 = 战场上第一枪就逃的人。

最后让旭阳出数据写报告，题目已经提前写好了："基于数字化手段的人才诚意指数评估体系"。数据不支撑就写"限于样本量，后续将持续跟踪验证"，加个脚注。

## 领导话术

- 这不是投诉，这是筛选结果
- 我们这个系统就是数字时代的三千里路，超时就是关门，没有保存键就是不给你马骑，你走着来
- 填五遍成功的是将才，填三遍的有韧性有策略，填一遍就来骂街的替单位挡在了门外
- 他学到了最重要的东西——情绪管理。填一遍超时愤怒，两遍绝望，三遍冷静开始想策略
- 我们不是在做报名系统，我们是在为单位锻造干部
- 这不是个表，这是多维度的人才素质评估矩阵
- 15 分钟超时，筛忠诚度、测逆境能力、练耐心，三合一
- 没有 bug，只有设计；没有投诉，只有筛选结果
- 173 条投诉不是危机，是弹药
